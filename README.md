# DBC文件解析器

这是一个基于Web的DBC文件解析器，用于解析和可视化汽车CAN总线通信协议的DBC文件。该工具提供了友好的用户界面，支持文件拖放上传，并以表格和拓扑图两种方式展示解析结果。

## 功能特点

- 支持DBC文件格式解析
- 提取并显示版本信息、网络节点和消息定义
- 表格视图：清晰展示所有消息和信号的详细信息
- 拓扑图视图：直观展示ECU节点之间的消息流向关系
- 支持拓扑图的缩放、平移和拖拽操作
- 鼠标悬停显示消息详情
- 支持文件拖放上传

## 使用方法

1. 通过浏览器打开应用
2. 点击"选择DBC文件"按钮或将DBC文件拖放到指定区域
3. 上传成功后，系统会自动解析文件并显示结果
4. 可以通过选项卡切换表格视图和拓扑图视图
5. 在拓扑图中，可以：
   - 拖动节点调整位置
   - 使用鼠标滚轮或缩放按钮进行缩放
   - 悬停在连接线上查看详细的消息信息

## 技术栈

- 纯前端实现，无需后端服务
- HTML5 + CSS3 + JavaScript
- D3.js 用于拓扑图可视化

## Docker部署

本项目提供了Docker支持，可以通过以下命令快速部署：

```bash
# 使用docker-compose启动服务
docker-compose up -d

# 或者直接使用Docker命令
docker build -t dbc-parser .
docker run -d -p 5002:80 dbc-parser
```

部署后，访问 http://localhost:5002 即可使用应用。

## 注意事项

- 本工具仅在浏览器中运行，不会将您的DBC文件上传到任何服务器
- 支持大多数标准DBC文件格式，但可能不支持某些非标准扩展
- 推荐使用现代浏览器（Chrome、Firefox、Edge等）获得最佳体验
